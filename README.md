# AutoClicker Pro

Продвинутый автокликер с глобальными горячими клавишами, настройкой интервала, выбором кнопки мыши и типов кликов. Дизайн в стиле Kaspersky.

## Возможности

### Основные
- **Мышь** — одинарный / двойной клик (левая, правая, средняя кнопка)
- **Клавиатура** — отправка нажатий клавиш через SendInput
- **Комбо-клавиши** — комбинации типа Ctrl+C, Shift+Tab и т.д.
- **Интервал** — от 1 мс до 60 секунд, переключение мс/CPS
- **Режимы** — Toggle (переключение) и Hold (удержание)
- **Рандомизация** — ±N% от интервала для обхода детекта
- **Лимит кликов** — автоостановка после заданного количества

### Привязка к окну
- Выбор окна из выпадающего списка
- Кликер работает только в выбранном окне через PostMessage
- Опция "Глобально" для работы в любом окне

### Пресеты
- Сохранение текущих настроек с именем
- Загрузка и удаление пресетов
- Отдельная вкладка для управления

### Горячие клавиши
- **F6** — старт/стоп (по умолчанию, переназначается)
- Горячая клавиша сброса счётчика
- Настройка через нажатие в интерфейсе

### Системный трей
- Сворачивание в трей при сворачивании окна
- Контекстное меню: Показать / Старт-Стоп / Выход
- Двойной клик по иконке — восстановить окно

### Автозапуск
- Регистрация в `shell:startup` для запуска с Windows

### Звуки и анимации
- Звуковые эффекты при запуске/остановке
- Пульс индикатора статуса
- Плавное переключение страниц (fade-in)
- Настройка вкл/выкл

## Требования

- Windows 10/11
- .NET 10.0 Runtime (или выше)
- Для Roblox: запуск от имени администратора

## Сборка

```bash

git clone https://github.com/dima-kovboi/AutoClicker.git
cd AutoClicker

dotnet build -c Release

# Запустить
dotnet run
```

## Настройки

### Главная
- Интервал (мс или CPS)
- Тип ввода: Мышь / Клавиатура
- Кнопка мыши: Левая / Правая / Средняя
- Тип действия: Одинарный / Двойной
- Режим: Переключение / Удержание
- Комбо-клавиши

### Настройки
- Горячие клавиши (Старт/Стоп, Сброс)
- Лимит кликов
- Рандомизация интервала
- Привязка к окну
- Звуки и анимации
- Сворачивать в трей
- Запускать с Windows

### Пресеты
- Создание / загрузка / удаление пресетов

## Технические детали

- **Мышь**: `mouse_event` API — совместимо со всеми приложениями
- **Клавиатура**: `SendInput` с скан-кодами — работает в Roblox и других защищённых приложениях
- **Комбо-клавиши**: Последовательное нажатие/отпускание клавиш через SendInput
- **Привязка к окну**: `PostMessage` — отправка сообщений конкретному окну
- **Хук клавиатуры**: `WH_KEYBOARD_LL` — перехват всех нажатий системой
- **Звуки**: Генерация WAV-файлов программно (синусоиды) без внешних зависимостей
- **Настройки**: JSON-файл в `%AppData%/AutoClicker/`
